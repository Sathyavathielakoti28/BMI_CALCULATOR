A simple and user-friendly **Body Mass Index (BMI) Calculator** developed using **Python** and **Tkinter**. This project calculates a user's BMI based on their weight and height, classifies the BMI into health categories, and provides a graphical user interface for easy interaction.

---

## 📌 Project Overview

The BMI Calculator helps users determine whether they are underweight, normal weight, overweight, or obese based on the standard BMI formula.

This project was developed as part of the **Oasis Infobyte Python Development Internship**.

- Command-Line BMI Calculator
- Graphical User Interface (GUI) using Tkinter

## 📐 BMI Formula

```
BMI = Weight (kg) / Height² (m²)
```

---

## 📊 BMI Categories

| BMI Range | Category |
|-----------|----------|
| Less than 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and above | Obese |
